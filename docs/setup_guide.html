🤖 สร้าง Telegram Bot ผ่าน @BotFather

358|

BotFather เป็นบอทของ Telegram ที่ใช้สร้างและจัดการบอทอื่นๆ

359| 360|

ขั้นตอน

361|
    362|
  1. เปิด Telegram และค้นหา @BotFather หรือกดลิงก์นี้: t.me/botfather
    2. 363|
  2. กด Start หรือพิมพ์ /start
    3. 364|
  3. พิมพ์ /newbot เพื่อสร้างบอทใหม่
    4. 365|
  4. BotFather จะถามชื่อบอท — พิมพ์ชื่อ เช่น Hydra Signal Bot
    5. 366|
  5. จากนั้นจะถาม username — ต้องลงท้ายด้วย bot เช่น YourHydraSignalBot
    6. 367|
  6. BotFather จะส่ง BOT TOKEN มาให้ — เก็บ Token นี้ไว้ในที่ปลอดภัย!
    7. 368|

🖥️ ตั้งค่า Hydra Signal Server

403|

เซิร์ฟเวอร์ที่รับสัญญาณจาก Master และกระจายไปยังลูกค้าผ่าน Telegram

404| 405|

ตัวเลือกที่ 1: ใช้ Docker (แนะนำ)

406| 
# 1. Clone โปรเจค
   407|git clone https://github.com/singhtofc-cell/Hydra-Trading-System.git
   408|cd Hydra-Trading-System
   409|
   410|# 2. แก้ไขไฟล์ .env
   411|cp .env.example .env
   412|nano .env
   413|# ใส่ HYDRA_TELEGRAM_BOT_TOKEN และ HYDRA_ADMIN_CHAT_ID
414| 415|

แก้ไขไฟล์ .env ให้มีเนื้อหาดังนี้:

416| 
HYDRA_TELEGRAM_BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
   417|HYDRA_ADMIN_CHAT_ID=123456789
   418|HYDRA_DATABASE_URL=postgresql://hydra:***@localhost:5432/hydra_trading
   419|HYDRA_SERVER_PORT=8788
420| 421| 
# 3. รันด้วย Docker Compose
   422|docker compose up -d
   423|
   424|# 4. ตรวจสอบว่าเซิร์ฟเวอร์ทำงาน
   425|curl http://localhost:8788/health
426| 427|
428| 💡 ผลลัพธ์ที่ได้: 429| 
{
   430|  "status": "healthy",
   431|  "version": "1.0.0",
   432|  "uptime": "0h 0m 15s",
   433|  "active_clients": 0,
   434|  "total_signals_dispatched": 0
   435|}
