PART 2

🤖 สร้าง Telegram Bot ผ่าน @BotFather

BotFather เป็นบอทของ Telegram ที่ใช้สร้างและจัดการบอทอื่นๆ

ขั้นตอน

  1. เปิด Telegram และค้นหา @BotFather หรือกดลิงก์นี้: t.me/botfather
  2. กด Start หรือพิมพ์ /start
  3. พิมพ์ /newbot เพื่อสร้างบอทใหม่
  4. BotFather จะถามชื่อบอท — พิมพ์ชื่อ เช่น Hydra Signal Bot
  5. จากนั้นจะถาม username — ต้องลงท้ายด้วย bot เช่น YourHydraSignalBot
  6. BotFather จะส่ง BOT TOKEN มาให้ — เก็บ Token นี้ไว้ในที่ปลอดภัย!
ภาพ: ตัวอย่างการสร้าง Bot ใหม่ผ่าน @BotFather — โปรดทำตามขั้นตอนในคู่มือ
⚠️ สำคัญ: Token ของคุณเป็นความลับสูงสุด! อย่าแชร์ให้ใครเด็ดขาด Token มีรูปแบบประมาณ: 1234567890:ABCdefGHIjklMNOpqrsTUVwxyz

ตั้งค่า Bot Commands

หลังจากได้ Token แล้ว พิมพ์คำสั่งต่อไปนี้ใน BotFather:

/setcommands

จากนั้นส่งข้อความนี้:

start - 🤖 เริ่มต้นใช้งาน
register - 📝 ลงทะเบียนบัญชี MT5
status - 📊 ดูสถานะปัจจุบัน
pause - ⏸️ หยุดรับสัญญาณชั่วคราว
resume - ▶️ เริ่มรับสัญญาณอีกครั้ง
report - 📈 ดูรายงานผลการเทรด
ภาพ: เอกสาร BotFather อย่างเป็นทางการจาก Telegram
PART 3

🖥️ ตั้งค่า Hydra Signal Server

เซิร์ฟเวอร์ที่รับสัญญาณจาก Master และกระจายไปยังลูกค้าผ่าน Telegram

ตัวเลือกที่ 1: ใช้ Docker (แนะนำ)

# 1. Clone โปรเจค
git clone https://github.com/singhtofc-cell/Hydra-Trading-System.git
cd Hydra-Trading-System

# 2. แก้ไขไฟล์ .env
cp .env.example .env
nano .env
# ใส่ HYDRA_TELEGRAM_BOT_TOKEN และ HYDRA_ADMIN_CHAT_ID

แก้ไขไฟล์ .env ให้มีเนื้อหาดังนี้:

HYDRA_TELEGRAM_BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
HYDRA_ADMIN_CHAT_ID=123456789
HYDRA_DATABASE_URL=postgresql://hydra:hydra_pass@localhost:5432/hydra_trading
HYDRA_SERVER_PORT=8788
# 3. รันด้วย Docker Compose
docker compose up -d

# 4. ตรวจสอบว่าเซิร์ฟเวอร์ทำงาน
curl http://localhost:8788/health
💡 ผลลัพธ์ที่ได้: 
{
  "status": "healthy",
  "version": "1.0.0",
  "uptime": "0h 0m 15s",
  "active_clients": 0,
  "total_signals_dispatched": 0
}

ตัวเลือกที่ 2: รันด้วย Python โดยตรง

# 1. Clone โปรเจค
git clone https://github.com/singhtofc-cell/Hydra-Trading-System.git
cd Hydra-Trading-System

# 2. ติดตั้ง dependencies
pip install -r requirements.txt

# 3. ตั้งค่า Environment
export HYDRA_TELEGRAM_BOT_TOKEN="1234567890:ABC..."
export HYDRA_ADMIN_CHAT_ID="123456789"

# 4. รัน Signal Server
python -m backend.signal_server
⚠️ หมายเหตุ: เซิร์ฟเวอร์ต้องรันตลอด 24/7 บนเครื่องที่เชื่อมต่ออินเทอร์เน็ตได้ หากใช้ Docker Compose ระบบจะ restart อัตโนมัติเมื่อเซิร์ฟเวอร์ล่ม
PART 4

🔑 ลงทะเบียน Telegram Bot Token

นำ Token ที่ได้จาก BotFather ไปตั้งค่าในระบบ

ขั้นตอน

  1. เปิดไฟล์ .env ในโปรเจค Hydra
  2. แทนที่ YOUR_BOT_TOKEN_HERE ด้วย Token จริง
  3. แทนที่ YOUR_ADMIN_CHAT_ID_HERE ด้วย Chat ID ของคุณ

วิธีหา Chat ID ของตัวเอง

# วิธีที่ 1: ใช้ @userinfobot
# ค้นหา @userinfobot ใน Telegram → กด Start
# Bot จะส่ง Chat ID ของคุณกลับมา

# วิธีที่ 2: ใช้ @getidsbot  
# ค้นหา @getidsbot → กด Start
# Bot จะแสดง ID, Username, และข้อมูลอื่นๆ
💡 Tips: เมื่อตั้งค่าเสร็จแล้ว ให้ Restart Server หรือ Docker Container เพื่อให้ Token ใหม่มีผลทันที
PART 5

📥 ติดตั้ง HydraCopyEA บน MT5

ติดตั้ง Expert Advisor ที่จะรับสัญญาณจาก Telegram และเทรดอัตโนมัติ

ขั้นตอนการติดตั้ง

  1. ดาวน์โหลดไฟล์ HydraCopyEA.ex5 จาก Admin หรือ compile จาก source
  2. เปิด MT5 → ไปที่เมนู File → Open Data Folder
  3. ไปที่โฟลเดอร์ MQL5\Experts\
  4. วางไฟล์ HydraCopyEA.ex5 ในโฟลเดอร์นี้
  5. ไปที่ MQL5\Include\ — วางไฟล์ HydraEA_Utils.mqh
  6. สร้างโฟลเดอร์ MQL5\Files\Hydra\
  7. สร้างไฟล์ bot_token.txt ในโฟลเดอร์ MQL5\Files\Hydra\
  8. เขียน Token ลงใน bot_token.txt (บรรทัดเดียว)
  9. Restart MT5 หรือกด Refresh ใน Navigator panel
  10. ลาก EA จาก Navigator ไปวางบน Chart ที่ต้องการ
🚨 คำเตือน: หาก MT5 ไม่ยอมให้ลาก EA ลง Chart:
  • ไปที่ Tools → Options → Expert Advisors
  • ✔ ติ๊ก "Allow Automated Trading"
  • ✔ ติ๊ก "Allow DLL imports" (จำเป็นสำหรับ WebRequest)
  • ✔ เพิ่ม https://api.telegram.org ใน "Allow WebRequest for listed URL"

การสร้างไฟล์ bot_token.txt

# เปิด Notepad บน VPS ของคุณ
# พิมพ์ Token ของคุณ (บรรทัดเดียวเท่านั้น!)
1234567890:ABCdefGHIjklMNOpqrsTUVwxyz

# บันทึกเป็น: C:\Users\[YourUser]\AppData\Roaming\MetaQuotes\Terminal\[InstanceID]\MQL5\Files\Hydra\bot_token.txt
💡 หา Instance ID ของ MT5: เปิด MT5 → File → Open Data Folder → จะเห็น path ที่มี Instance ID ต่อท้าย เช่น ...\Terminal\A1B2C3D4E5F6\

PART 7

📝 ลงทะเบียนบัญชีกับ Telegram Bot

แจ้งให้ระบบรู้ว่าคุณเป็นลูกค้าและตั้งค่าการรับสัญญาณ

ขั้นตอน

  1. เปิด Telegram และค้นหา Bot ที่สร้างไว้ เช่น @YourHydraSignalBot
  2. กด Start หรือพิมพ์ /start
  3. ระบบจะแสดงเมนูต้อนรับพร้อมปุ่มต่างๆ
  4. พิมพ์คำสั่งลงทะเบียน:
/register 12345678 USD 1.0

โดยที่:

  • 12345678 = เลขบัญชี MT5 ของคุณ
  • USD = ประเภทสกุลเงิน (USD หรือ USC)
  • 1.0 = Lot multiplier (1.0 สำหรับ USD, 100 สำหรับ USC)
💡 ตัวอย่าง: 
/register 87654321 USC 100
สำหรับบัญชี Cent USC — Master ส่ง 0.01 lot → คุณจะได้ 1.00 lot

เมนูคำสั่งทั้งหมด

คำสั่งการทำงาน
/startแสดงเมนูต้อนรับ
/register [acc] [cur] [mult]ลงทะเบียนบัญชี
/statusดูสถานะการรับสัญญาณ
/pauseหยุดรับสัญญาณชั่วคราว
/resumeเริ่มรับสัญญาณอีกครั้ง
/reportดูรายงานผลวันนี้
/report weeklyดูรายงานผลประจำสัปดาห์
PART 8

🧪 ทดสอบการทำงาน

ตรวจสอบว่าระบบทำงานถูกต้องก่อนเริ่มเทรดจริง

1. ตรวจสอบ Health Endpoint

เปิด Terminal หรือ Browser แล้วไปที่:

http://[YOUR_SERVER_IP]:8788/health

2. ส่ง Signal ทดสอบ

curl -X POST http://localhost:8788/api/v1/signal/send \
  -H "Content-Type: application/json" \
  -d '{
    "signal_type": "BUY",
    "symbol": "XAUUSD",
    "entry_price": 2350.50,
    "sl_price": 2330.00,
    "tp_prices": [2370.00],
    "lot_multiplier": 1.0,
    "risk_percent": 1.0,
    "source": "TEST"
  }'

3. ตรวจสอบว่า EA รับสัญญาณได้

เปิด MT5 → ไปที่ Expert Tab (ใน Terminal panel ด้านล่าง) ควรเห็นข้อความ:

🤖 Hydra Copy EA initialized
📡 New signal received: sig_0001
✅ Signal executed: sig_0001
📍 Order placed: Ticket=12345678 Lot=0.01 SL=2330.00 TP1=2370.00

4. ตรวจสอบ Telegram

เปิด Telegram Bot ของคุณ — ควรได้รับข้อความ:

🟢 HYDRA SIGNAL #sig_0001
🟢 BUY XAUUSD
━━━━━━━━━━━━━━━━━━━━━━━
📍 Entry: 2350.50
🛡️ SL: 2330.00
💰 TP1: 2370.00
🚨 ถ้าไม่ได้รับสัญญาณ:
  • ตรวจสอบ EA Journal Tab — มี error หรือไม่
  • ตรวจสอบว่า Bot Token ถูกต้อง
  • ตรวจสอบว่า MT5 เปิด Auto Trading อยู่
  • ตรวจสอบว่า WebRequest ไปยัง api.telegram.org ได้

ประเภทเวลาเนื้อหา

👑 Admin Summaryทุกวัน 23:55 UTCสรุปรวมลูกค้าทุกคน (เฉพาะ Admin)

เปิดเบราว์เซอร์ไปที่:

http://[YOUR_SERVER_IP]:8501

PART 10

🔧 การแก้ไขปัญหาเบื้องต้น

ปัญหาที่พบบ่อยและวิธีการแก้ไข

❌ EA ไม่ยอมติดบน Chart

สาเหตุ: MT5 ปิด Auto Trading ไว้
วิธีแก้: Tools → Options → Expert Advisors → ติ๊ก Allow Automated Trading

❌ EA ไม่ได้รับสัญญาณจาก Telegram

สาเหตุ: WebRequest ถูกบล็อก
วิธีแก้: Tools → Options → Expert Advisors → เพิ่ม https://api.telegram.org ใน Allow WebRequest

❌ ได้รับข้อความ "Bot token not found"

สาเหตุ: ไม่พบไฟล์ bot_token.txt
วิธีแก้: สร้างไฟล์ ...\MQL5\Files\Hydra\bot_token.txt และใส่ Token

❌ Signal Server ไม่ยอมเริ่ม

สาเหตุ: Port 8788 ถูกใช้งานแล้ว
วิธีแก้: เปลี่ยน port ใน .env หรือ config.yaml: 
HYDRA_SERVER_PORT=8789

❌ Docker Container เริ่มไม่ได้

สาเหตุ: Port ขัดแย้งหรือ Token ไม่ถูกต้อง
วิธีแก้: 
docker compose logs hydra-api    # ดู Error log
docker compose restart             # Restart ทั้งหมด

❌ สลิปเพจสูงเกินไป

สาเหตุ: ราคาตลาดเคลื่อนที่เร็วกว่าที่ EA จะเทรดได้
วิธีแก้: เพิ่ม InpSlippage ใน Input Parameters 
InpSlippage = 50   # จากเดิม 30 points

❌ EA ส่งรายงานไม่กลับมา

สาเหตุ: InpSendReports = false หรือ WebRequest ถูกบล็อก
วิธีแก้: ตรวจสอบ Input → InpSendReports = true
APPENDIX

📋 ภาคผนวก: ค่า Parameter แยกตามประเภทบัญชี

ตารางเปรียบเทียบค่าที่แนะนำสำหรับบัญชีแต่ละประเภท

💵 USD Standard Account

Parameterค่าเหตุผล
InpRiskPercent1.0%ความเสี่ยงมาตรฐาน 1% ต่อเทรด
InpLotMultiplier1.0Lot เท่ากับ Master โดยตรง
InpMaxLotCap5.0จำกัด Lot สูงสุดไม่เกิน 5.0
InpMaxDailyLoss5.0%หยุดเทรดเมื่อขาดทุนถึง 5%
InpMaxDrawdown15.0%ปิดออเดอร์ทั้งหมดเมื่อ DD 15%
ตัวอย่าง: Master ส่ง BUY EURUSD Lot 0.01 → คุณจะได้ BUY EURUSD Lot 0.01 ด้วย

💴 USC Cent Account

Parameterค่าเหตุผล
InpRiskPercent1.0%ความเสี่ยงมาตรฐาน 1% ต่อเทรด
InpLotMultiplier100เพื่อแปลง Cent → Standard lot
